beatoraja用スキンテンプレートエンジン【ModernChicTemplateEngine】
******************************************************************

ダウンロードしていただきありがとうございます。
このスキンはスキン作成をしやすいように設計したbeatoraja用のスキンテンプレートです。
グローバル変数MAIN、CUSTOMに様々な情報を追加してありオプション番号やタイマー番号などを調べやすいようにしています。
詳しくはブログにて書いていくつもりです。（かなりゆっくりだとは思いますが
	
★改変スキンや二次配布等について
	スキンに含まれるデータについてはどのように使用して頂いても構いませんが
	本スキンそのものを二次配布することは禁止とさせて頂きます。
	改変したスキンを公開する場合はreadme等で原作者名（KASAKO）の記述をお願いします。

    当テンプレートを使用しているかを判断するためにheader.lua、nameプロパティの(MCTE)表記は消さないようお願いします。

	また改変スキンを使い何か問題が発生した場合はその責任は負いかねます。
	信頼出来ない場合はDLするのは控えたほうがいいです。（結構大事）

このスキンがスキン作成のちょっとした手助けに慣れれば嬉しい限りです。

本スキンでは表示フォントに「Mgen+」(http://jikasei.me/font/mgenplus/) を使用しています。
